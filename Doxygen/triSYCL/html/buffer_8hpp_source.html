480 /** Set destination of buffer data on destruction
481
482 The finalData points to the host memory to which, the outcome of all
483 the buffer processing is going to be copied to.
484
485 This is the final pointer, which is going to be accessible after the
486 destruction of the buffer and in the case where this is a valid
