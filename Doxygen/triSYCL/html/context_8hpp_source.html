33/** SYCL context
34
35 The context class encapsulates an OpenCL context, which is implicitly
36 created and the lifetime of the context instance defines the lifetime
37 of the underlying OpenCL context instance.
38
39 On destruction clReleaseContext is called.
40
41 The default context is the SYCL host context containing only the SYCL
42 host device.
43
44 \todo The implementation is quite minimal for now.
45*/
