Detailed Description

A Boost.Fiber pool implemented on a std::thread pool.

The use case is for circuit emulation when there are a lot of fibers launched at the beginning and they have to run concurrently.

The fiber executors can use several std::thread.

It allows the execution of some callable and return some future for later shepherding.
