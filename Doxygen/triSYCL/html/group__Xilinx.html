template<typename ValueType, std::size_t Size, typename PartitionType = partition::none>
struct trisycl::vendor::xilinx::partition_array< ValueType, Size, PartitionType >

Define an array class with partition feature.

Since on FPGA, users can customize the memory architecture in the system and within the CU. Array partition help us to partition single array to multiple memories that can be accessed simultaneously getting higher memory bandwidth.

Parameters
ValueTypeis the type of element.
Sizeis the size of the array.
PartitionTypeis the array partition type: cyclic, block, and complete. The default type is none.
