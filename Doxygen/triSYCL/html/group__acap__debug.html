template<typename Geography>
class trisycl::vendor::xilinx::acap::debug::bsp_checker< Geography >

Verify that an SPMD program has a bulk-synchronous behaviour across all the tiles of the CGRA.

If there are some discrepancies, display the tile id which is too late or too early.
