Detailed Description

template<typename T, int Dimensions>
template<std::size_t N>
struct trisycl::mixin::accessor< T, Dimensions >::track_index< N >

Proxy object to transform an expression like accessor[i1][i2][i3] into the implementation mdpsan(i1,i2,i3) one index at a time.

It gathers intermediate [index] to finally call the mdspan indexing operator once they are all available

\parameter[in] N is the number of indices which can be stored in this proxy
