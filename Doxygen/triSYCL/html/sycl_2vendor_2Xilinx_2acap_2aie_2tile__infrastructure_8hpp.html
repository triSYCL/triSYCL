Detailed Description

The basic AI Engine homogeneous tile, with the common infrastructure to all the tiles, i.e.

independent of x & y coordinates, but also from the tile program itself.

This tile can be seen as the SYCL subdevice of the raw CGRA SYCL device to run elemental functions.

This is owned by the device, so for example the AXI stream switch configuration and packet can survive to some program changes.
